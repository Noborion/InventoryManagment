# INSTRUÇÕES DE USO DOS PROGRAMAS DE TESTE

## Conteúdo

Este diretório contém programas de teste abrangente para o gerenciador de estoque "Sneaker King".

### Arquivos Principais:

1. **teste_completo.c** - Código fonte dos testes para gerenciador_estoque.c
2. **teste_completo.exe** - Executável dos testes

### Documentos de Relatório:

1. **RELATORIO_TESTES.md** - Relatório detalhado com análise completa
2. **SUMARIO_TESTES.txt** - Sumário executivo com recomendações
3. **LISTA_COMPLETA_TESTES.txt** - Lista com todos os 24 testes
4. **INSTRUCOES.txt** - Este arquivo

---

## Como Executar os Testes

### Opção 1: Executar o programa já compilado

```bash
# Para testar versão (24 testes)
.\teste_completo.exe
```

### Opção 2: Recompilar e executar

```bash
# Para recompilar
gcc -o teste_completo teste_completo.c
.\teste_completo.exe
```

## O Que Cada Teste Valida

### Teste de Cadastro:
- Armazenamento correto de dados
- Atribuição automática de códigos
- Múltiplos cadastros sequenciais

### Teste de Atualização:
- Modificação de quantidade
- Restrição a produto válido
- Valores altos e zero

### Teste de Validação:
- Rejeição de código inválido
- Proteção contra índice negativo
- Validação de limite superior

### Teste de Capacidade:
- Limite máximo respeitado
- Rejeição após limite atingido
- Contagem correta de produtos

### Teste de Estoque Baixo:
- Detecção automática
- Exibição de alerta visual
- Limiar de 5 unidades (básico) / 10 (avançado)

### Teste de Quantidades Extremas:
- Valor zero aceito
- Valor negativo (comportamento registrado)
- Valor muito alto (999)

### Teste de Tamanhos:
- Armazenamento de tamanhos
- Intervalo suportado

### Teste de Strings:
- Strings vazias
- Strings longas
- Caracteres especiais

### Teste de Múltiplos Produtos:
- Cadastro de vários produtos
- Atualização em lote
- Visualização formatada

### Teste de Busca:
- Busca por marca
- Busca por modelo
- Busca por tamanho

### Teste de Exclusão:
- Marcação como inativo
- Não aparição em buscas
- Preservação de dados

### Teste de Relatório:
- Cálculo de totais
- Estatísticas (min/max)
- Detecção de alertas

## Interpretação de Avisos

Durante a execução, você pode ver:

```
[AVISO] Sistema permite valores negativos
```

Isso significa que o sistema **não** valida o valor, mas permite que seja armazenado.
Este é um comportamento registrado para documentar limitações.
